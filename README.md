# ParkiPay — Government Parking Fee Collection System (Tanzania)

A mobile-first solution for fair, transparent, and auditable parking fee collection across government and public parking areas in Tanzania.

## Tech Stack

| Layer       | Technology                        |
|-------------|-----------------------------------|
| Mobile App  | React Native + Expo               |
| Backend API | Django REST Framework             |
| Database    | PostgreSQL (Supabase)             |
| Deployment  | Docker + Render                   |
| Auth        | JWT (djangorestframework-simplejwt) |
| Messaging   | Africa's Talking SMS Gateway      |

## Project Structure

```
parkipay/
├── frontend/        # React Native + Expo mobile app
├── backend/         # Django REST API
├── proposal/        # Project summary and documentation
└── project_plan/    # Agile sprint plans (Scrum)
```
