## User Stories

**As a** development team,
**I want** comprehensive automated tests covering all critical paths,
**So that** we can deploy with confidence and catch regressions quickly.

**Acceptance Criteria:**
- [ ] Backend test coverage ≥ 80% overall (measured with `coverage.py`)
- [ ] All API endpoints have integration tests (happy path + error cases)
- [ ] Critical paths have 100% test coverage:
      — Login / lockout
      — Plate lookup (valid, invalid, bad format)
      — Control number generation (new, duplicate, post-expiry)
      — Expiry background task
      — SMS dispatch (mock Africa's Talking)
- [ ] All tests pass on CI (GitHub Actions)
- [ ] Frontend: key user flows tested with Jest + React Native Testing Library

**Tasks:**
- [ ] Run `coverage run manage.py test && coverage report`
- [ ] Identify and fill gaps to reach 80% threshold
- [ ] Write integration tests using `APITestCase` for all endpoints
- [ ] Mock Africa's Talking and SendGrid in tests (no real API calls)
- [ ] Write E2E test for full flow: login → plate lookup → generate bill → history
- [ ] Frontend: test `LoginForm`, `PlateInput`, `VehicleCard`, `BillHistoryCard`
- [ ] Generate and commit HTML coverage report to `docs/coverage/`

### US-602 | Security Audit & Hardening
**As a** government IT team,
**I want** the system reviewed against common vulnerabilities,
**So that** officer credentials and citizen data are protected.

**Acceptance Criteria:**
- [ ] No hardcoded secrets in codebase (verified by `git log` and `trufflesecurity/trufflehog`)
- [ ] All endpoints require authentication (no accidentally public routes)
- [ ] SQL injection: all queries use Django ORM (no raw SQL with user input)
- [ ] Rate limiting applied to: login (5/min), plate lookup (30/min), generate (10/min)
- [ ] CORS restricted to known origins only (Expo app + admin domain)
- [ ] Django `DEBUG = False` in production
- [ ] `SECRET_KEY` is ≥ 50 chars random value from environment
- [ ] Supabase RLS policies verified on all tables
- [ ] HTTPS enforced (redirect HTTP → HTTPS on Render)

**Tasks:**
- [ ] Run `bandit -r .` (Python security linter) — fix all HIGH severity issues
- [ ] Install `django-ratelimit` and apply to sensitive views
- [ ] Audit all `urlpatterns` — verify every view has `IsAuthenticated`
- [ ] Verify no raw SQL with user input (grep for `.raw(`, `cursor.execute(`)
- [ ] Set `ALLOWED_HOSTS` to Render domain only
- [ ] Set `SECURE_SSL_REDIRECT = True`, `SESSION_COOKIE_SECURE = True`
- [ ] Verify Supabase RLS is ON for all tables
- [ ] Run TruffleHog on full git history to scan for leaked secrets
- [ ] Document security decisions in `docs/security.md`
